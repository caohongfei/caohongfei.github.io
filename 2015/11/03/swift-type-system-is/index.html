4. Array上的is操作符

那么怎么判断一个定义为Array<Animal>的值实际上是不是Array<Cat>呢？答案是使用is操作符。有趣的地方在于，Swift的is操作符对于数组会采取逐个元素动态判断的方式。它既不是根据编译时数组的类型来决定，也不使用数组变量的dynamicType值。
